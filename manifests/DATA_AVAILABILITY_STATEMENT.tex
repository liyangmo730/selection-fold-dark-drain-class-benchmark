\section*{Data Availability}

The numerical tables, figure-generation scripts, manuscript source, and
compressed $S_8$ comparison data used in this work are provided in the curated
public reproducibility release accompanying the manuscript:
\url{https://github.com/liyangmo730/selection-fold-dark-drain-class-benchmark}.
The repository contains the data needed to reproduce the figures, benchmark
tables, dense $z=0$ matter-power window check, and compressed-amplitude
comparison. It does not include the full exploratory development history, a
full CLASS source tree, or a DES/KiDS/HSC/Planck likelihood pipeline. If an
archival DOI is added later, the DOI should be cited for the versioned release.
